# Movies App

This web application allows users to discover movies, upcoming movies, TV shows, and trending content. The app is powered by the Movie Database (TMDB) API.

# Key Features

Movies: Each movie includes title, release date, overview, etc. Users can add movies to their favorites list.

Upcoming Movies: Upcoming Movies include the same information available for movies. Users can add movies to their favorites list.

TV Show Showcase: TV Shows include title, release date, overview, etc. Users can add TV Shows to their favorites list.

Unified Favorites: In this app favorites for movies, upcoming movies and TV shows are unified under the same favorites tab. 

Trending: The app has a tab for trending movies. Users can add trending movies to their favorites.

Pagination: Most tabs in this app include pagination in the bottom of each page, for a smooth browsing experience.

Sorting: The favorites tab sorts movies in alphabetical order based on their origin, i.e., movies, TV shows and Upcoming Movies are sorted separately.
